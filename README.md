# Reference Net Editor
Web based reference net editor.

# Instructions

## Usage

```
window.onload = () => new window.drawing(canvasId);
```

## Hotkeys

| Key   | Action          |
| :---: | :-------------- |
|  v    | Selection Tool  |
|  r    | Rectangle Tool  |
|  c    | Circle Tool     |
|  p    | Place Tool      |
|  t    | Transition Tool |
